# IPL Data Analysis (2022–2026)

This project focuses on analyzing ball-by-ball IPL datasets from 2022 to 2026 using Python. The goal is to extract meaningful insights about players, teams, and match performance through data analysis and visualization.

## 📌 Project Overview

The dataset contains detailed delivery-level data such as:

* Batter and bowler information
* Runs scored on each ball
* Wickets and match events
* Over-by-over match progress

This project performs basic data operations and exploratory data analysis (EDA) to understand trends and patterns in IPL matches.

## ⚙️ Operations Performed

* Data Loading using Pandas
* Data Cleaning (handling missing values)
* Filtering and Sorting

## 📊 Key Insights

* Top run scorers across seasons
* Most successful bowlers (wickets)
* Runs scored per over
* Team performance distribution

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
